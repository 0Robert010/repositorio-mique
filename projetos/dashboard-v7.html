Este case foi separado como projeto em andamento para mostrar a evolução técnica planejada: formulário, autenticação, banco de dados e painel de gestão.

Problema

Uma loja precisa visualizar produtos, estoque e vendas com rapidez. Fazer isso só em mensagens ou planilhas dificulta acompanhamento e tomada de decisão.

Solução planejada

Criar um painel web com cards de métricas, lista de produtos, formulário de cadastro e área protegida por login simples.

Roadmap técnico

  • Primeira versão: HTML, CSS, JavaScript e dados simulados.
  • Segunda versão: persistência com SQLite ou MySQL.
  • Terceira versão: login, CRUD e API simples.
