SmartWaste

Projeto UPx da FACENS para apoiar a gestão de resíduos no campus, com monitoramento simulado de lixeiras, alertas e dashboard responsivo.

Resultado
Dashboard com pontos simulados, status por cor e documentação do projeto.

Problema

A coleta de resíduos pode perder eficiência quando não existe uma visão rápida de quais lixeiras estão cheias, em alerta ou dentro do limite.

Solução

O SmartWaste apresenta pontos de coleta, níveis de ocupação, status visual e atualização simulada em JavaScript para mostrar como um sistema IoT poderia funcionar.

O que eu fiz

  • Estruturei a interface web e a apresentação do problema/solução.
  • Criei dashboard com cards, filtros, barras de progresso e estados visuais.
  • Organizei documentação e deploy para o projeto ficar demonstrável.

Próximos passos

  • Conectar sensores reais ou API.
  • Salvar histórico em banco de dados.
  • Adicionar autenticação para equipe de coleta.
